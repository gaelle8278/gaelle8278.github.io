<div>
 Personnaliser Gnome, Chrome et autres outils linux.
<!--more-->
Gnome :
  <ul>
      <li>Installer Gnome Tweak</li>
      <li>Activer et paramétrer les extensions d'intéret : Dash to Panel, Menu application, Dash to Dock ...</li>
      <li>Activer extension User Theme</li>
      <li>Installation Pling-Store</li>
      <li>Via Pling-Store installer thème gnome et pack d'icones provenenant de <a href="https://www.gnome-look.org/browse/cat/">gnome-look.org</a></li>
  </ul>
<p>
Chrome : <br>
Trouver des themes (sur devianart notamment) puis ajouter le theme en glissant le fichier .crx dans chome://extensions 
</p>

<p>
Eclipse : <br>
Créer/Trouver des color scheme et les ajouter soit via le .xml grâce à l'extension Eclipse color Theme (incluse dans Color IDE Pack), soit via le .epf puis File > Import... > General > Preferences .
</p>
Source : https://mkyong.com/eclipse/how-to-change-eclipse-theme/
